You have 3 options:

OPTION 1: SSH (Recommended)
---------------------------
1. Set up SSH key if you haven't:
   ssh-keygen -t ed25519 -C "teddytennant@icloud.com"
   cat ~/.ssh/id_ed25519.pub

2. Add key to GitHub:
   https://github.com/settings/keys

3. Change remote to SSH:
   git remote set-url origin git@github.com:teddytennant/rusttorch.git

4. Push:
   git push origin main

OPTION 2: Personal Access Token
--------------------------------
1. Create token at:
   https://github.com/settings/tokens

2. Push with token:
   git push https://<YOUR-TOKEN>@github.com/teddytennant/rusttorch main

OPTION 3: GitHub CLI
---------------------
1. Install and authenticate:
   gh auth login

2. Push:
   git push origin main
